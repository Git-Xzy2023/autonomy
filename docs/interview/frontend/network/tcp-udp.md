# TCP 和 UDP 的区别

## 简介

**TCP**（传输控制协议）：面向连接的、可靠的、基于字节流的传输层通信协议，为不可靠的互联网络提供可靠的端到端字节流。

**UDP**（用户数据报协议）：无连接的传输层协议，提供面向事务的简单不可靠信息传送服务。

## UDP 的特点

- 无连接，减少开销和时延
- 尽最大努力交付，不保证可靠
- 面向报文，不合并也不拆分
- 没有拥塞控制
- 支持一对一、一对多、多对一、多对多
- 首部开销小，仅 8 字节

## 对比表格

| 方面 | TCP | UDP |
|------|-----|-----|
| 连接 | 面向连接 | 无连接 |
| 可靠性 | 可靠传输 | 不可靠传输 |
| 传输方式 | 面向字节流 | 面向报文 |
| 拥塞控制 | 有 | 没有 |
| 流量控制 | 有 | 没有 |
| 首部开销 | 20 字节 | 8 字节 |
| 应用场景 | 文件传输、邮件、网页浏览 | 实时通信、视频直播、DNS 查询 |
