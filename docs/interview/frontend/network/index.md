# 常见的 HTTP 方法有哪些？

| 方法 | 作用 | 备注 |
|------|------|------|
| GET | 获取资源 | 目前网络请求大部分使用 GET 请求 |
| POST | 传输资源 | 如登录等提交操作 |
| PUT | 更新资源 | 如上传头像等 |
| DELETE | 删除资源 | 如删除评论等 |
| HEAD | 获取报文首部 | 和 GET 类似，但不返回报文实体主体部分。用于确认 URL 有效性及资源更新日期时间 |
| OPTIONS | 查询支持的方法 | 查询指定 URL 能够支持哪些方法 |
| PATCH | 部分更新资源 | 对 PUT 的补充，用于对已知资源进行局部更新 |
