# 总结：CSS 学习路径

**初级（基础）：**

1. 选择器、盒模型、定位（position）
2. 浮动（float）和清除浮动
3. 常见布局技巧（居中、两栏、三栏）

**中级（进阶）：**

1. Flexbox
2. CSS Grid
3. 响应式设计和媒体查询
4. CSS 动画（transition、transform、animation）
5. BFC、层叠上下文

**高级（专家）：**

1. CSS 变量和自定义属性
2. 新特性（aspect-ratio、:has()、container queries）
3. 性能优化（回流、重绘、GPU 加速）
4. CSS 架构（BEM、CSS Modules、CSS-in-JS）
5. 浏览器兼容性和工程化
