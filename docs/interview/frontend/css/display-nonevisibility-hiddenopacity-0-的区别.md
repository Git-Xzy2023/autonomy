# display: none、visibility: hidden、opacity: 0 的区别？

| 特性         | `display: none` | `visibility: hidden` | `opacity: 0` |
| ------------ | --------------- | -------------------- | ------------ |
| 是否占据空间 | ❌ 不占据       | ✅ 占据              | ✅ 占据      |
| 是否可交互   | ❌ 不可点击     | ❌ 不可点击          | ✅ 可点击    |
| 是否触发回流 | ✅ 触发         | ❌ 不触发            | ❌ 不触发    |
| 是否触发重绘 | ✅ 触发         | ✅ 触发              | ✅ 触发      |
| 子元素影响   | 子元素也隐藏    | 子元素也隐藏         | 子元素也透明 |
| 过渡动画     | ❌ 不支持       | ❌ 不支持            | ✅ 支持      |
