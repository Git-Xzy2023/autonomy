# 如何实现一个自适应的正方形？

```css
/* 方案一：使用 aspect-ratio（现代方案） */
.square {
  aspect-ratio: 1 / 1;
  width: 100%;
}

/* 方案二：使用 padding-top（兼容方案） */
.square {
  position: relative;
  width: 100%;
  padding-top: 100%; /* 1:1 比例 */
}
.square-content {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}

/* 方案三：使用 vw/vh */
.square {
  width: 30vw;
  height: 30vw;
}
```
