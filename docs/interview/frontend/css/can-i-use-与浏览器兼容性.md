# Can I Use 与浏览器兼容性

**使用 `caniuse.com` 查询 CSS 特性的浏览器支持情况。**

**常见 CSS 特性兼容性：**

| 特性              | Chrome  | Firefox | Safari   | IE            |
| ----------------- | ------- | ------- | -------- | ------------- |
| Flexbox           | ✅ 29+  | ✅ 28+  | ✅ 9+    | ✅ 11（部分） |
| CSS Grid          | ✅ 57+  | ✅ 52+  | ✅ 10.1+ | ❌            |
| CSS 变量          | ✅ 49+  | ✅ 31+  | ✅ 9.1+  | ❌            |
| `clamp()`         | ✅ 79+  | ✅ 75+  | ✅ 13.1+ | ❌            |
| `aspect-ratio`    | ✅ 88+  | ✅ 89+  | ✅ 15+   | ❌            |
| `:has()`          | ✅ 105+ | ✅ 121+ | ✅ 15.4+ | ❌            |
| Container Queries | ✅ 105+ | ✅ 110+ | ✅ 16+   | ❌            |
