# 什么是条件类型？语法是什么？

条件类型是 TS 类型系统中的 **if-else**，语法：

```ts
T extends U ? X : Y
```

读作：**如果 T 可以赋值给 U，那么取 X，否则取 Y**。

**简单示例：**

```ts
type IsString<T> = T extends string ? true : false;
type A = IsString<"hello">; // true
type B = IsString<123>; // false
type C = IsString<string>; // true
```
