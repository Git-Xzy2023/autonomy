# 浏览器组成

## 对浏览器的理解

浏览器的主要功能是将用户选择的 web 资源呈现出来。浏览器分为两部分：
- **shell**：浏览器的外壳（菜单、工具栏等）
- **内核**：浏览器的核心，基于标记语言显示内容的程序或模块

## 浏览器内核

浏览器内核主要分成两部分：
1. **渲染引擎**：显示所请求的内容
2. **JS 引擎**：解析和执行 JavaScript

## 常见浏览器内核

| 内核 | 浏览器 | 特点 |
|------|--------|------|
| Trident | IE | 早期市场份额大，对 W3C 标准支持不好 |
| Gecko | Firefox | 功能强大丰富，消耗资源多 |
| Presto | Opera（旧版） | 浏览速度最快，兼容性稍差 |
| Webkit | Safari | 浏览速度较快，对代码容错性不高 |
| Blink | Chrome、Opera（新版） | Webkit 分支，当前广泛使用 |

## 浏览器主要组成部分

1. **用户界面**：地址栏、前进/后退按钮、书签等
2. **浏览器引擎**：在用户界面和呈现引擎之间传送指令
3. **呈现引擎**：负责显示请求的内容
4. **网络**：用于网络调用
5. **用户界面后端**：绘制基本控件
6. **JavaScript 解释器**：解析和执行 JavaScript 代码
7. **数据存储**：持久层
