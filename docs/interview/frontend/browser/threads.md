# 浏览器渲染进程有哪些线程？

浏览器的渲染进程总共有五种线程：

| 线程 | 职责 |
|------|------|
| **GUI 渲染线程** | 负责渲染浏览器页面，解析 HTML/CSS，构建 DOM 树、CSSOM 树、渲染树和绘制页面。与 JS 引擎线程互斥 |
| **JS 引擎线程** | 负责处理 Javascript 脚本程序。一个 Tab 页中任何时候只有一个 JS 引擎线程在运行 JS |
| **事件触发线程** | 属于浏览器而不是 JS 引擎，用来控制事件循环 |
| **定时器触发线程** | setInterval 与 setTimeout 所在线程。定时时间不能小于 4ms |
| **异步 HTTP 请求线程** | XMLHttpRequest 连接后通过浏览器新开一个线程请求 |

## 进程之间的通信方式

- **管道通信**：只能单向通信，只能血缘关系的进程通信
- **消息队列通信**：在消息队列中添加/读取消息
- **信号量通信**：实现进程之间的互斥与同步
- **信号通信**：操作系统通知进程发生了预先规定好的事件
- **共享内存通信**：多个进程共享一个给定的存储区
