# SemVer（语义化版本）

格式：`主版本.次版本.修订号`

- **主版本**：破坏性变更（不兼容的 API 改动）
- **次版本**：向后兼容的新增功能
- **修订号**：向后兼容的 bug 修复

**版本范围符号：**

| 符号        | 含义                   | 示例            | 匹配             |
| ----------- | ---------------------- | --------------- | ---------------- |
| `^`         | 不改变最左边的非零数字 | `^1.2.3`        | `>=1.2.3 <2.0.0` |
| `~`         | 次版本不变             | `~1.2.3`        | `>=1.2.3 <1.3.0` |
| `*`         | 任意版本               | `*`             | 所有             |
| `>` `<` `=` | 大小比较               | `>1.2.3`        | 大于 1.2.3       |
| `-`         | 区间                   | `1.2.3 - 1.4.0` | 闭区间           |
