# 手写 `模板渲染引擎`（简易版）

```js
function render(tpl, data) {
  // 把 {{ name }} 替换成 data.name
  return tpl.replace(/\{\{\s*(\w+)\s*\}\}/g, (_, k) => data[k] ?? "");
}
render("Hello, {{ name }}! You are {{ age }}.", { name: "Alice", age: 30 });
// 'Hello, Alice! You are 30.'
```
