# 数组方法 `at()` 是什么？为什么要引入它？（ES2022）

**`arr.at(i)` 支持负索引**，和 Python 的 `arr[-1]` 一样：

```js
const arr = [10, 20, 30, 40];
arr.at(-1); // 40        （等价于 arr[arr.length - 1]，但更直观）
arr.at(-2); // 30
arr.at(0);  // 10
arr.at(99); // undefined
```

字符串同样有 `"hello".at(-1)`。之前要写 `arr[arr.length - 1]` 很啰嗦，`arr[-1]` 在 JS 里又被当作属性名而不是索引。
