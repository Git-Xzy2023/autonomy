# 逻辑赋值运算符 `&&=` / `||=` / `??=`（ES2021）

```js
let a = 1, b = 0, c = null;

a ||= 100; // 1    (a 已经 truthy，不变)
b &&= 200; // 0    (b 是 falsy，不变)
c ??= 300; // 300  (c 是 null，赋值)

// 等价于：
// a = a || 100;
// b = b && 200;
// c = c ?? 300;
```
