# 对象字面量增强有哪些写法？

```js
const name = "Alice";
const age = 18;

// 1) 简写属性（shorthand properties）
const person = { name, age };       // { name: name, age: age }

// 2) 简写方法（shorthand methods）
const obj = {
  greet() { return "hi"; },         // 等价于 greet: function() { ... }
};

// 3) 计算属性名（computed property names）
const key = "age";
const user = {
  [key]: 18,                        // { age: 18 }
  ["get" + "Name"]() { return this.name; },
};
```
