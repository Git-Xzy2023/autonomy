# 用 ES6+ 语法重构下面这段老代码：

```js
// 老代码
function greet(user) {
  user = user || {};
  var name = user.name || "Guest";
  var age = user.age || 18;
  return "Hello " + name + ", you are " + age + " years old.";
}
```

**重构后**：

```js
function greet({ name = "Guest", age = 18 } = {}) {
  return `Hello ${name}, you are ${age} years old.`;
}
```

> 要点：对象解构 + 默认值 + 整体参数默认 `{}` 防止传 `undefined` 时抛错 + 模板字符串。
