# Nginx 学习指南

> Nginx 是高性能的 HTTP 服务器与反向代理，广泛用于负载均衡、API 网关、静态资源服务。本章节系统整理 Nginx 配置、反向代理、负载均衡与性能优化。

---

## 一、为什么学习 Nginx

- 🚀 **高并发**：事件驱动架构，单机数万并发
- 🌐 **反向代理**：隐藏后端服务，统一入口
- ⚖️ **负载均衡**：多后端分流，横向扩展
- 🔒 **HTTPS**：SSL/TLS 终结，安全传输
- 📦 **静态资源**：高性能静态文件服务
- 🛠️ **运维必备**：前端部署、微服务网关的基础设施

## 三、Nginx 核心能力

```
客户端请求
   ↓
Nginx
├── 静态资源服务（HTML/CSS/JS/图片）
├── 反向代理（转发到后端服务）
├── 负载均衡（多后端分流）
├── HTTPS 终结（SSL 证书）
├── Gzip 压缩
├── 缓存
└── 访问控制（限流、黑白名单）
```

---

## 四、推荐学习路线

1. **基础**：安装、配置文件结构、location 匹配规则
2. **反向代理**：proxy_pass、upstream、负载均衡算法
3. **优化**：worker 进程、keepalive、gzip、缓存、HTTPS
