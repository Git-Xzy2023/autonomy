# MyBatis

> MyBatis 是半自动 ORM 框架，相比 JPA 更灵活，SQL 手写可控。本章涵盖映射文件、动态 SQL、关联查询、缓存。

## MyBatis vs JPA

| 特性     | MyBatis       | JPA           |
| -------- | ------------- | ------------- |
| SQL      | 手写          | 自动生成      |
| 灵活性   | 高            | 低            |
| 学习成本 | 中            | 低            |
| 复杂查询 | 强            | 弱            |
| 国内使用 | 主流          | 较少          |
