# Spring 框架

> Spring 是 Java 企业级开发的事实标准，核心是 IoC（控制反转）和 AOP（面向切面编程）。本章涵盖 IoC 容器、AOP、Spring MVC、事务管理。

## 章节导航

| 章节                                 | 主题  | 内容                       |
| ------------------------------------ | ----- | -------------------------- |
| [01 IoC 容器](/java/spring/01-ioc/)  | IoC   | Bean 生命周期、依赖注入     |
| [02 AOP](/java/spring/02-aop/)        | AOP   | 切面、通知、切入点          |
| [03 Spring MVC](/java/spring/03-mvc/)| MVC   | 请求流程、参数绑定、拦截器 |
| [04 事务](/java/spring/04-transaction/)| 事务 | 声明式事务、传播行为        |

## Spring 生态

```
Spring Framework   基础框架（IoC / AOP / MVC / 事务）
├── Spring Boot      快速开发（自动配置）
├── Spring Security  安全（认证授权）
├── Spring Data      数据访问（JPA / Redis / MongoDB）
├── Spring Cloud     微服务（网关 / 注册 / 配置）
├── Spring Batch     批处理
└── Spring Integration 集成
```
