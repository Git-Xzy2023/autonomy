# Java 基础

> Java 基础是后端开发的根基，涵盖语法、面向对象、集合框架、并发编程、JVM 原理。本章从语法到 JVM，系统梳理 Java 核心知识。

| [02 面向对象](/java/basic/02-oop/)            | OOP      | 类、继承、多态、接口           |
| [03 集合框架](/java/basic/03-collections/)    | 集合     | List / Set / Map 及源码         |

| [06 新特性](/java/basic/06-new-features/)     | 现代 Java | Lambda、Stream、Record、模块化 |

## 学习建议

1. **语法与 OOP 是地基**：不要跳过，后续所有内容都建立其上。
2. **集合与并发是核心**：面试高频、工程高频，务必手写代码验证。
3. **JVM 是分水岭**：区分初级与中高级，理解 GC 和类加载。
