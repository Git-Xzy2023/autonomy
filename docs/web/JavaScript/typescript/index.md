## 为什么学习 TypeScript？

TypeScript 是 JavaScript 的超集，添加了可选的静态类型检查和基于类的面向对象编程。

### 🎯 TypeScript 的核心优势

1. **类型安全**：编译时发现错误，减少运行时 bug
2. **智能提示**：强大的 IDE 支持，提升开发效率
3. **代码维护**：清晰的类型定义，便于团队协作
4. **渐进式 adoption**：可以逐步迁移现有 JavaScript 项目

### 📊 学习路径

```
基础类型 → 接口与类 → 泛型 → 高级类型 → 装饰器 → 最佳实践
```

### 🎓 你将学到什么

通过这个 TypeScript 学习模块，你将掌握：

- ✅ TypeScript 的基础语法和类型系统
- ✅ 接口、类、泛型的使用
- ✅ 高级类型技巧（条件类型、映射类型等）
- ✅ 与 JavaScript 的互操作
- ✅ TypeScript 项目配置和最佳实践
