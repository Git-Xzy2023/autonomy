# JavaScript 技术学习指南

本章节涵盖 Web 开发中常见的脚本技术，包括原生 **JavaScript**、**TypeScript** 和 **ES6+ 新特性**。

## 为什么学习 JavaScript？

JavaScript 是 Web 前端开发的三大基石之一。无论是做一个简单的交互效果、还是复杂的企业级应用，都离不开 JavaScript。掌握好 JavaScript 可以帮助你：

- ✅ **构建交互**：实现丰富的用户交互和动态效果
- ✅ **数据处理**：处理和管理应用程序的数据
- ✅ **跨平台开发**：不仅限于浏览器，还可用于 Node.js、移动端、桌面端
- ✅ **框架开发**：学习 React、Vue 等现代框架的基础

## 各模块说明

### 📜 JavaScript

JavaScript 是所有 Web 脚本的基础。从最基础的语法到异步编程、面向对象，它是你必须掌握的核心技能。

**推荐学习路径：**

1. [JS 基础](/web/JavaScript/javascript/01-basics/) — 变量、数据类型、运算符、流程控制
2. [函数与作用域](/web/JavaScript/javascript/02-functions/) — 函数定义、闭包、this 指向
3. [对象与原型](/web/JavaScript/javascript/03-objects/) — 对象操作、原型链、继承
4. [异步编程](/web/JavaScript/javascript/04-async/) — Promise、async/await、事件循环
5. [数组方法与高阶函数](/web/JavaScript/javascript/06-arrays/) — map、filter、reduce
6. [模块化](/web/JavaScript/javascript/07-modules/) — ES Modules、CommonJS
7. [最佳实践](/web/JavaScript/javascript/08-best-practices/) — 代码风格、性能优化

### 📘 TypeScript

TypeScript 是 JavaScript 的超集，添加了可选的静态类型检查和基于类的面向对象编程。

**快速上手：**

- [TypeScript 入门](/web/JavaScript/typescript/)

### ✨ ES6+ 新特性

ES6+ 引入了许多革命性的新特性，极大提升了开发效率和代码质量。

**快速上手：**

- [ES6+ 入门](/web/JavaScript/es6/)

## 技术对比速查表

| 特性         | JavaScript | TypeScript           |
| ------------ | ---------- | -------------------- |
| **类型系统** | 动态类型   | 静态类型（可选）     |
| **编译检查** | 运行时检查 | 编译时检查           |
| **工具支持** | 基础       | 丰富（IDE 智能提示） |
| **学习曲线** | 较低       | 中等                 |
| **项目规模** | 中小型     | 中大型               |

## 学习建议

1. **新手入门**：先学习 [JS 基础](/web/JavaScript/javascript/01-basics/)，理解脚本编程的核心概念。
2. **项目实战**：学完基础后，尝试实现一些小型项目（待办事项、计算器等）。
3. **进阶提升**：学习 TypeScript 提升代码质量，掌握 ES6+ 提高开发效率。
