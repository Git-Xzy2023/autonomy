# 创建小程序

> 本章将从零开始，带你完成微信小程序的注册、开发者工具安装、项目创建，并了解小程序的基本概念和版本体系。

---

## 一、小程序简介

### 1.1 什么是小程序？

小程序是一种不需要下载安装即可使用的应用，它实现了"触手可及"的梦想，用户扫一扫或搜一下即可打开应用。

```
┌─────────────────────────────────────────────────┐
│                   小程序特点                      │
├─────────────────────────────────────────────────┤
│  ✅ 无需下载安装，触手可及                         │
│  ✅ 用完即走，无需卸载                            │
│  ✅ 体验接近原生 App                              │
│  ✅ 可在微信内便捷传播                            │
│  ✅ 开发成本低于原生 App                          │
│  ✅ 支持跨平台（通过跨端框架）                     │
└─────────────────────────────────────────────────┘
```

### 1.2 小程序 vs H5 vs 原生 App

| 特性         | 小程序          | H5             | 原生 App       |
| ------------ | --------------- | -------------- | -------------- |
| **开发成本** | 中              | 低             | 高             |
| **体验**     | 接近原生        | 一般           | 最佳           |
| **安装**     | 无需安装        | 无需安装       | 需下载安装     |
| **入口**     | 微信内          | 浏览器         | 桌面图标       |
| **能力**     | 丰富（设备API） | 受限           | 最完整         |
| **审核**     | 需要            | 不需要         | 需要           |
| **分发**     | 微信生态        | URL 分发       | 应用商店       |
| **离线能力** | 支持            | 需 PWA         | 完全支持       |

### 1.3 主流小程序平台

| 平台         | 开发者     | 技术栈           |
| ------------ | ---------- | ---------------- |
| **微信小程序** | 腾讯       | WXML/WXSS/JS     |
| **支付宝小程序** | 蚂蚁集团   | AXML/ACSS/JS     |
| **百度小程序** | 百度       | SWAN/CSS/JS      |
| **字节小程序** | 字节跳动   | TTML/TTSS/JS     |
| **QQ 小程序** | 腾讯       | 与微信类似       |
| **快应用**   | 手机厂商联盟 | 类 Vue 语法      |

> 💡 **建议**：初学者从微信小程序入手，生态最完善、文档最齐全、用户量最大。本章后续内容以微信小程序为例。

## 二、注册微信小程序账号

### 2.1 注册流程

1. **访问注册页面**：打开 [https://mp.weixin.qq.com/wxopen/waregister](https://mp.weixin.qq.com/wxopen/waregister)

2. **选择账号类型**：选择"小程序"

```
┌─────────────────────────────────────┐
│         微信公众平台账号类型          │
├─────────────────────────────────────┤
│                                     │
│   📱 服务号    📱 订阅号             │
│   （面向企业）  （面向个人/媒体）     │
│                                     │
│   📲 小程序    🎮 微信小游戏         │
│   （本章重点）  （游戏开发）          │
│                                     │
└─────────────────────────────────────┘
```

3. **填写邮箱信息**：使用一个未注册过微信公众平台、未绑定过个人微信的邮箱

4. **邮箱激活**：登录邮箱查收激活邮件，点击激活链接

5. **信息登记**：
   - **个人主体**：填写身份证信息，扫码绑定管理员微信
   - **企业主体**：填写企业信息、营业执照、对公账户等

6. **注册完成**：获得小程序的 AppID

### 2.2 账号类型对比

| 类型     | 个人  | 企业   | 政府   | 媒体   | 其他组织 |
| -------- | ----- | ------ | ------ | ------ | -------- |
| **能否注册** | ✅    | ✅     | ✅     | ✅     | ✅       |
| **支付能力** | ❌    | ✅     | ✅     | ✅     | ✅       |
| **开放服务类目** | 受限  | 完整   | 完整   | 完整   | 完整     |
| **审核速度** | 较快  | 1-7天  | 1-7天  | 1-7天  | 1-7天    |

> ⚠️ **注意**：个人主体小程序无法使用微信支付能力，如果需要支付功能，必须注册企业主体。

### 2.3 获取 AppID

AppID 是小程序的唯一标识，开发时必须使用。

1. 登录 [微信公众平台](https://mp.weixin.qq.com)
2. 进入「开发」→「开发管理」→「开发设置」
3. 可以看到 **AppID（小程序ID）**

```
AppID 示例：wx1234567890abcdef
```

> ⚠️ **重要**：AppID 是敏感信息，不要提交到公开仓库。AppSecret 更是机密，切勿泄露！

## 三、安装微信开发者工具

### 3.1 下载

访问官方下载页面：[https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

根据操作系统选择对应版本：

| 操作系统 | 下载版本           |
| -------- | ------------------ |
| **Windows** | Windows 64 位      |
| **macOS** | macOS（Intel / Apple Silicon） |
| **Linux** | Linux 64 位        |

> 💡 **建议**：下载**稳定版（Stable Build）**，预发布版（RC）可能存在 Bug。

### 3.2 安装

- **Windows**：双击 `.exe` 安装包，按提示完成安装
- **macOS**：将 `.dmg` 中的「微信开发者工具」拖入「应用程序」文件夹
- **Linux**：解压 `.tar.gz`，运行其中的启动脚本

### 3.3 首次启动

1. 使用注册时绑定的管理员微信**扫码登录**

2. 登录后看到项目列表界面

### 3.4 开发者工具界面介绍

```
┌──────────────────────────────────────────────────────────────┐
│  菜单栏（文件、编辑、项目、设置、工具、微信开发者工具）        │
├──────────┬───────────────────────────┬───────────────────────┤
│          │                           │                       │
│  资源    │      代码编辑区            │     预览/调试区        │
│  管理器  │      （编辑代码）          │                       │
│          │                           │  ┌─────────────────┐  │
│  - 页面  │                           │  │                 │  │
│  - 组件  │                           │  │   模拟器        │  │
│  - 资源  │                           │  │   （实时预览）   │  │
│          │                           │  │                 │  │
│          │                           │  └─────────────────┘  │
│          │                           │                       │
│          │                           │  Console / Network /  │
```

**核心面板说明**：

| **Wxml**      | 查看 WXML 结构和样式                       |
| **Sensor**    | 模拟地理位置、加速度等传感器               |

## 四、创建第一个小程序项目

### 4.1 新建项目

1. 打开微信开发者工具，点击「+」号或「新建项目」

2. 填写项目信息：

```
项目目录：/Users/yourname/projects/my-first-miniprogram
AppID：填入你的 AppID（或选择"测试号"）
项目名称：my-first-miniprogram
开发模式：小程序
后端服务：不使用云服务（初学先选这个）
模板选择：JavaScript - 基础模板
```

3. 点击「确定」，项目创建完成

### 4.2 项目目录结构

创建后，项目目录如下：

```
my-first-miniprogram/
├── pages/                  # 页面目录
│   ├── index/              # 首页
│   │   ├── index.js        # 页面逻辑
│   │   ├── index.json      # 页面配置
│   │   ├── index.wxml      # 页面结构
│   │   └── index.wxss      # 页面样式
│   └── logs/               # 日志页
│       ├── logs.js
│       ├── logs.json
│       ├── logs.wxml
│       └── logs.wxss
├── utils/                  # 工具函数
│   └── util.js
├── app.js                  # 小程序入口逻辑
├── app.json                # 小程序全局配置
├── app.wxss                # 全局样式
├── project.config.json     # 项目配置（工具相关）
├── sitemap.json            # 搜索索引配置
└── .gitignore
```

### 4.3 运行项目

项目创建后会**自动编译**并在模拟器中运行，你会看到一个欢迎页面和日志页面。

**手动操作**：

- 修改代码后，按 `Ctrl+S`（macOS: `Cmd+S`）保存会自动重新编译
- 点击工具栏的「编译」按钮可手动触发编译
- 点击「预览」可生成二维码，用手机微信扫码在真机预览

### 4.4 第一个修改：Hello World

修改 `pages/index/index.wxml`：

```xml
<!--index.wxml-->
<view class="container">
  <view class="hello">
    <text>Hello, 小程序！</text>
  </view>
  <view class="user">
    <text>我的第一个小程序</text>
  </view>
</view>
```

修改 `pages/index/index.wxss`：

```css
/* index.wxss */
.container {
  display: flex;
  flex-direction: column;
  align-items: center;
  padding: 100rpx 0;
}

.hello {
  font-size: 48rpx;
  font-weight: bold;
  color: #07c160;
  margin-bottom: 40rpx;
}

.user {
  font-size: 32rpx;
  color: #888;
}
```

保存后，模拟器会实时刷新显示效果。

### 4.5 添加一个新页面

1. 在 `app.json` 的 `pages` 数组中添加页面路径：

```json
{
  "pages": [
    "pages/index/index",
    "pages/logs/logs",
    "pages/about/about"
  ],
  "window": {
    "backgroundTextStyle": "light",
    "navigationBarBackgroundColor": "#07c160",
    "navigationBarTitleText": "我的小程序",
    "navigationBarTextStyle": "white"
  }
}
```

2. 保存后，开发者工具会**自动创建** `pages/about/` 目录及四个文件

3. 编辑 `pages/about/about.wxml`：

```xml
<!--about.wxml-->
<view class="about">
  <view class="title">关于我们</view>
  <view class="content">这是一个学习用的小程序</view>
</view>
```

4. 在首页添加跳转按钮 `pages/index/index.wxml`：

```xml
<view class="container">
  <view class="hello">
    <text>Hello, 小程序！</text>
  </view>
  <button type="primary" bindtap="goAbout">去关于页</button>
</view>
```

5. 在 `pages/index/index.js` 添加跳转逻辑：

```javascript
Page({
  goAbout() {
    wx.navigateTo({
      url: '/pages/about/about'
    })
  }
})
```

保存后，点击首页按钮即可跳转到关于页。

## 五、小程序的版本体系

### 5.1 三个版本

```
┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│   开发版      │───▶│   体验版      │───▶│   正式版      │
│              │    │              │    │              │
│ 开发者工具中  │    │ 上传代码后    │    │ 审核通过后    │
│ 预览/真机调试 │    │ 体验成员可用  │    │ 所有用户可用  │
└──────────────┘    └──────────────┘    └──────────────┘
```

| 版本     | 获取方式               | 可见范围           | 用途               |
| -------- | ---------------------- | ------------------ | ------------------ |
| **开发版** | 开发者工具中运行       | 开发者本人         | 日常开发调试       |
| **体验版** | 上传代码后在后台设置   | 体验成员（最多15人）| 内测、产品验收     |
| **正式版** | 提交审核通过后发布     | 所有微信用户       | 线上运营           |

### 5.2 版本管理流程

```
开发调试 ──▶ 上传代码 ──▶ 设为体验版 ──▶ 提交审核 ──▶ 发布上线
  │                          │             │             │
  │                          │             │             │
  ▼                          ▼             ▼             ▼
开发版                      体验版        审核中        正式版
```

### 5.3 体验成员管理

1. 登录微信公众平台
2. 进入「管理」→「成员管理」
3. 在「体验成员」中添加微信好友
4. 体验成员扫码后可在微信中打开体验版

## 六、小程序的版本号规范

小程序版本号遵循语义化版本规范：

```
主版本号.次版本号.修订号
   1   .  2  .  3
```

| 版本号     | 何时升级             | 示例                     |
| ---------- | -------------------- | ------------------------ |
| **主版本** | 不兼容的 API 修改    | 1.0.0 → 2.0.0            |
| **次版本** | 向下兼容的功能新增   | 1.0.0 → 1.1.0            |
| **修订号** | 向下兼容的问题修复   | 1.0.0 → 1.0.1            |

> 💡 上传代码时需要填写版本号和项目备注，建议规范填写便于回溯。

## 七、开发前的准备清单

开始正式开发前，建议完成以下准备：

- [ ] 注册小程序账号，获取 AppID
- [ ] 安装微信开发者工具
- [ ] 创建项目并成功运行
- [ ] 了解小程序的版本体系
- [ ] 阅读小程序[官方文档](https://developers.weixin.qq.com/miniprogram/dev/framework/)
- [ ] 准备好设计稿和接口文档（如有）
- [ ] 配置服务器域名（如需网络请求）

### 7.1 服务器域名配置

小程序的网络请求必须使用 HTTPS，且域名需在小程序后台配置白名单。

1. 登录微信公众平台
2. 进入「开发」→「开发管理」→「开发设置」→「服务器域名」
3. 添加合法域名：

```
request 合法域名：https://api.example.com
socket 合法域名：wss://api.example.com
uploadFile 合法域名：https://upload.example.com
downloadFile 合法域名：https://download.example.com
```

> ⚠️ **注意**：
> - 每月可修改 50 次域名配置
> - 域名必须经过备案（国内服务器）
> - 必须使用 HTTPS/WSS
> - 不能使用 IP 地址或 localhost

### 7.2 开发阶段绕过域名校验

开发阶段可以在开发者工具中临时关闭域名校验：

- 点击工具栏「详情」→「本地设置」→ 勾选「不校验合法域名...」

> ⚠️ 此设置仅在开发者工具中生效，真机预览和正式环境仍需配置合法域名。
