## 为什么选择 Zustand

```
┌─────────────────────────────────────────┐
│           Zustand 优势                  │
├─────────────────────────────────────────┤
│                                         │
│  ✅ 极简 API                            │
│  ✅ 无需 Provider                       │
│  ✅ 无需 action/reducer                 │
│  ✅ TypeScript 友好                     │
│  ✅ 体积小（~1KB）                      │
│  ✅ 支持中间件                          │
│  ✅ 可组合                              │
│                                         │
└─────────────────────────────────────────┘
```
