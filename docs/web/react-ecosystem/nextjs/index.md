## 为什么选择 Next.js

```
┌─────────────────────────────────────────┐
│           Next.js 优势                  │
├─────────────────────────────────────────┤
│                                         │
│  ✅ SSR（服务端渲染）                   │
│  ✅ SSG（静态生成）                     │
│  ✅ ISR（增量静态再生）                 │
│  ✅ 文件系统路由                        │
│  ✅ API 路由                            │
│  ✅ 图片优化                            │
│  ✅ 字体优化                            │
│  ✅ 内置 CSS 支持                       │
│  ✅ TypeScript 支持                     │
│                                         │
└─────────────────────────────────────────┘
```
